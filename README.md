VPP: Virtual Protest Protocol (v1.0)
2D Avatar-based Digital Assembly System / 2Dアバターによる仮想デモ・集会システム
VPP is an open-source protocol designed to visualize collective will while maintaining individual safety. 本プロジェクトは、デジタル時代の新しい抗議の作法（プロトコル）をオープンソースで確立・共有するための提唱です。

1. Core Concept / プロジェクトの核心
[EN] The heart of this project lies in the realization of "Virtual Demos and Rallies" where individuals can participate from home using 2D avatars. It transforms small, individual voices into a visible "crowd" in digital space.

[JP] 本プロジェクトの核心は、**「家庭から2Dアバターを使って参加できる仮想デモ・集会」**の実現にあります。一人ひとりの小さな声をデジタル空間上で目に見える「群衆」へと変え、リスクを最小限に抑えながら集団の意思を可視化します。

2. Technical Specifications / 実装スペック
2.1 2D Avatar Interface / 2Dアバター・インターフェース
Avatar System / アバターシステム

[EN] Provide dozens of copyright-cleared avatars (formal, comical, heroes, animals, ethnic costumes).

[JP] 著作権をクリアした数十種類のアバターを提供（真面目、コミカル、擬人化動物、民族衣装など）。

Action & Message / モーションと主張

[EN] 3-4 stages of animation or still images, followed by a "Protest Speech Bubble" (up to 80 characters) on the final frame.

[JP] 3〜4段階のモーションアニメーション又は静止画を表示し、最後の画像に吹き出し形式で「主張テキスト（80字以内）」を表示。

Handle Names / ハンドルネーム

[EN] Max 10 characters (real names prohibited), displayed above the avatar.

[JP] 10文字以内（本名不可）、アバターの頭上に表示。

2.2 Visualization & Statistics / 参加・統計機能
Real-time Poll / リアルタイム集計

[EN] Visualizes the count of "Pros/Cons" through a real-time bar display.

[JP] 議題に対する「賛成・反対」数をリアルタイムでバー表示。

Crowd Unit Management / 群衆のユニット管理

[EN] Participants are managed in "cells" of 50-100 units to form two massive visual crowds (Pros vs. Cons) while reducing rendering load.

[JP] 参加者を50〜100人単位の「セル（細胞）」として扱い、描画負荷を軽減しつつ、賛成・反対の「二つの巨大な群衆」を構成。

Attribute Integration / 属性データの反映

[EN] User attributes (Gender, Age, Region) are instantly reflected in the Pros/Cons survey functions.

[JP] 選択された属性（性別・年齢層・居住地域）を、賛成・反対のアンケート機能へリアルタイムに反映。

3. National Sovereignty & Safety (Voice-of-Nation Rule) / 主権維持と安全性の原則
Domestic Focus & Local Protests / 国内限定

[EN] Each site (e.g., voice-of-japan.org) is strictly for residents of that nation. Virtual rallies are limited to actual domestic locations and issues. Using foreign plazas for domestic protests is prohibited.

[JP] 各サイトはその国の在住者専用です。バーチャルデモの舞台も国内の実在する場所と問題に限定されます（例：日本版で他国の広場を使用することは禁止）。

Non-Interference / 他国干渉の禁止

[EN] By restricting participation and locations to domestic boundaries, we protect the community from cross-border public opinion manipulation and cyber threats.

[JP] 参加者と場所を国内に限定することで、他国からの世論操作やサイバー攻撃からコミュニティを守ります。

International Issues / 国際問題は別枠

[EN] Global topics will be hosted on a separate platform with distinct international protocols.

[JP] 国際的なトピックについては、別のプロトコルに基づいた専用サイトで別途提供します。

4. Operational Models / 運営モデル
Standard Model (Non-Profit) / 日本・標準モデル

[JP] 「日本の声（voice-of-japan.org）」は連帯し非営利（NPO）方式で運営されます。

US Model (Hybrid/For-Profit) / 米国展開（Will of America）

[EN] To avoid naming conflicts and ensure sustainability, the US version may adopt for-profit or hybrid models through corporate sponsorships and media data licensing.

[JP] 「Will of America」は、VOAとの名称重複を避けつつ、企業スポンサーやメディアへのデータ提供を通じて持続可能性を確保するための営利・ハイブリッドモデルの採用を検討しています。
